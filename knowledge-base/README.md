# Conflict Tracker Knowledge Base

This folder is written for OSINT analysts (amateur or professional) who are not software engineers.

Use this order:

1. `getting-started.md`
2. `use-cases.md`
3. `ui-walkthrough.md`

## What this helps you do

- Start the system locally
- Load recent Iran strike/intercept reporting
- Read the map, timeline, and event details
- Capture local analyst notes safely in your browser
