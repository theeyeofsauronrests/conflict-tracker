# React Best Practices

Use this skill for React component design and review.

## Checklist
- Keep components focused and composable.
- Lift shared state only as far as required.
- Derive data instead of duplicating state.
- Keep side effects isolated with clear dependencies.
- Prefer explicit props contracts over hidden coupling.
- Separate presentational components from data orchestration.
