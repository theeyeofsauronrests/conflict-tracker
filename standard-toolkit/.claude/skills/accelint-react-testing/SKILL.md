# React Testing

Use this skill for React test implementation.

## Checklist
- Test behavior from user-visible outcomes.
- Prefer Testing Library queries over implementation details.
- Mock only expensive or external integrations.
- Keep smoke tests for top-level render paths.
- Assert accessibility labels/roles for key controls.
- Keep fixtures minimal and representative.
