# TS Performance

Use this skill for performance-sensitive TypeScript code.

## Checklist
- Avoid repeated heavy parsing in hot paths.
- Minimize intermediate allocations in pipelines.
- Batch remote/database operations.
- Cache stable computed results where appropriate.
- Prefer O(n) passes with clear memory bounds.
- Measure before and after optimization changes.
