# README Writer

Use this skill when creating or updating repository README files.

## Checklist
- State purpose and architecture briefly.
- List prerequisites and environment variables.
- Provide local run/test/build commands.
- Document extension workflows (plugins/modules).
- Include security and deployment notes.
- Keep instructions copy-paste friendly.
