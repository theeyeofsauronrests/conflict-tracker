# TS Audit All

Use this skill for full-repo audits.

## Audit Order
1. Correctness and behavioral regressions.
2. Security and data exposure risks.
3. Performance bottlenecks and scaling concerns.
4. Type safety and contract drift.
5. Test coverage gaps and missing edge cases.
6. Documentation mismatches.

## Output Format
- Findings ordered by severity.
- Include file and line references.
- Add concrete remediation for each issue.
